# Hypatia Review Center

Production-ready Next.js site for Hypatia Review Center, a criminology review center in the Philippines.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- React Hook Form
- Zod
- Lucide Icons
- Swiper

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Routes

- `/`
- `/about`
- `/programs`
- `/admissions`
- `/faculty`
- `/news`
- `/gallery`
- `/testimonials`
- `/faq`
- `/contact`
- `/student-portal`
- `/privacy`
- `/terms`

## Production Checks

```bash
npm run build
npm run lint
```

The project is configured for static generation and Vercel deployment.
